## 🎯 立即执行：Phase 12.1 实盘环境搭建

### 目标
搭建真实交易环境，为实盘交易做准备。

### 步骤
1. 申请 Futu 真实交易账户权限
2. 配置真实交易 API 密钥
3. 搭建独立的实盘环境（与模拟盘隔离）
4. 配置实盘专用风控参数（更严格）
5. 设置紧急平仓机制

### 风控参数（实盘）
```yaml
# 实盘风控（比模拟盘更严格）
real_trading:
  max_single_position_pct: 0.05    # 5% (模拟盘 10%)
  stop_loss_pct: 0.05              # 5% (模拟盘 8%)
  take_profit_pct: 0.15            # 15% (模拟盘 20%)
  max_daily_loss_pct: 0.02         # 2% (模拟盘 3%)
  max_drawdown_pct: 0.10           # 10% (模拟盘 15%)
  initial_capital: 10000           # 1 万元测试
```
